Micro:bit MIDI

Create a device that is able to be used in live performances or while working on a DAW (Digital Audio Interface) to control an element like an EQ. Thea device would be able to set values based on its position and orientation. This would be done by using MIDI as the interface.

DESCRIPTION

The sensors chosen
Gyroscope and the A B Buttons

 The outputs chosen
MIDI Serial

How your project works
There's a library in Makecode and it outputs midi serial data through the usb connection after that we used a driver called Hairless Midi which parses serial usb data (the usb connected to microbit), next we need a way to output that data so a DAW can access it. Usually macs already have a midi software installed so it's easier. But for the output we downloaded a virtual midi controler by Tobias Erichsen and passed the parsed data to that virtual line of MIDI. Then in FL Studio (a music production software or DAW) we enabled the midi input and then we specified what we wanted to do with the midi values there. In this case it was assigned to an EQ to show the effects showing visiually. But any value can be used. any knob or in any VST Plugin.
